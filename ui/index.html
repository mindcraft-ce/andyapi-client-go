About AndyAPI Client

This client connects your local AI models to the AndyAPI server via WebSocket, allowing them to be accessed through a unified API endpoint.

Configure your local OpenAI-compatible API (such as Ollama, LM Studio, or vLLM), scan for available models, and connect to start serving requests.
